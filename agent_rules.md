## Rules
- 変更作業の完了時は、ユーザーから明示的に停止指示がない限り、必ず `git add` → `git commit` → `git push` まで実行すること。
- PRタイトル・PR本文・PRコメントは必ず日本語で作成すること。
- PR本文は `.github/PULL_REQUEST_TEMPLATE.md` に必ず準拠すること。
- コミットメッセージは日本語で作成すること。
- UniTaskが返り値の関数がコールされ、_キーワードで破棄されている場合はForget()を使うように促してください。同時に提案してください。
- privateフィールドは _ + camelCase,publicフィールドはPascalCaseです。
